About Treatment Packages

Treatment packages are pre-configured plans that include:

  • Pre-set number of treatment sessions
  • Fixed total cost with potential savings
  • Specific therapy type recommendations

How to use:

  1. Select a package that matches patient needs
  2. Review the session count and cost details
  3. Package will auto-fill relevant form fields
  4. Modify values if needed before saving
